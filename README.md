# ☕ Advanced Java Training Repository

Welcome to the official repository for **Advanced Java Training**. This space is dedicated to daily coding practices, assignments, and weekly assessments designed to master enterprise-level Java development.

---

## 📂 Repository Structure & Navigation

This repository is organized using **branches** to separate daily tasks from assessments. Please navigate to the specific branches below to access the code and questions.

| Branch Name | Content Description | Access |
| :--- | :--- | :--- |
| **`Assignments`** | 📝 Daily practice topics, coding exercises, and questions. | [View Branch](/tree/Assignments) |
| **`Weekly-Assessments`** | 📊 Code solutions and questions for weekly evaluations. | [View Branch](/tree/Weekly-Assessments) |

---

## 🚀 How to Access the Code

### Option A: Via GitHub UI
1. Click on the **Branch** dropdown menu at the top left of this repository (next to the main file list).
2. Select either `Assignments` or `Weekly-Assessments`.
3. You can also click the **"View Branch"** links in the table above.

### Option B: Via Git CLI
If you have cloned the repository to your local machine, switch branches using the following commands:

```bash
# For Daily Practices
git checkout Assignments

# For Weekly Assessments
git checkout Weekly-Assessments
